## Задача: 
    ```
    Написать программу, которая из имеющегося массива строк формирует массив из строк.
    Длина которых меьше либо равна 3 символа.
    Первоначальный массив можно ввести с клавиатуры, либо задать на старте.
    ```
## Решение задачи:
```
Первым этапом в задаче было составление массива, также вывод его на экран;
Второй этап проверка по всей длине массива количества элементов меньше либо равных трем и выделение для них места в новом массиве;
Третий этап создание массива используя результаты предыдущей проверки и вывод нового массива в результат.
```
